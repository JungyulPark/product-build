## 1. Executive Summary

### 1.1 프로젝트 개요
K-Fortunes 사이트의 사주 결과 페이지에 "나와 사주가 같은 K-스타" 기능을 보너스 콘텐츠로 추가하여 바이럴 공유를 극대화하는 프로젝트.

### 1.2 핵심 가치 제안
- **자기 발견:** "나와 같은 운명의 연예인이 누구지?"
- **공유 욕구:** "나 BTS 정국이랑 사주가 같대!"  
- **대화 주제:** 친구들끼리 비교 → 바이럴 루프
- **K-Culture 시너지:** 글로벌 K-Pop/K-Drama 팬덤 활용

### 1.3 성공 지표 (KPI)
| 지표 | 목표 | 측정 방법 |
|------|------|----------|
| 공유율 | 결과 페이지 방문자의 15%+ | Share 버튼 클릭 추적 |
| 바이럴 계수 | 1.2+ (1명이 1.2명 유입) | UTM 파라미터 추적 |
| 체류 시간 | 평균 3분+ | Google Analytics |
| 재방문율 | 20%+ | GA 리텐션 리포트 |

### 2.2 매칭 알고리즘

#### 2.2.1 사주 요소별 가중치
```javascript
const MATCHING_WEIGHTS = {
  dayMaster: 40,      // 일간 (가장 중요: 본질적 성격)
  monthBranch: 25,    // 월지 (사회적 성향)
  yearStem: 15,       // 년간 (세대/시대 에너지)
  yearBranch: 10,     // 년지 (띠)
  hourPillar: 10      // 시주 (있는 경우만)
};
```

#### 2.2.2 매칭 점수 계산
```javascript
function calculateMatchScore(userSaju, celebritySaju) {
  let score = 0;
  
  // 일간 매칭 (40점)
  if (userSaju.dayMaster === celebritySaju.dayMaster) {
    score += 40;
  } else if (isSameElement(userSaju.dayMaster, celebritySaju.dayMaster)) {
    score += 20; // 같은 오행이면 부분 점수
  }
  
  // 월지 매칭 (25점)
  if (userSaju.monthBranch === celebritySaju.monthBranch) {
    score += 25;
  } else if (isSameTriangle(userSaju.monthBranch, celebritySaju.monthBranch)) {
    score += 12; // 삼합이면 부분 점수
  }
  
  // 년간 매칭 (15점)
  if (userSaju.yearStem === celebritySaju.yearStem) {
    score += 15;
  }
  
  // 년지(띠) 매칭 (10점)
  if (userSaju.yearBranch === celebritySaju.yearBranch) {
    score += 10;
  }
  
  // 시주 매칭 (10점) - 둘 다 있는 경우만
  if (userSaju.hourPillar && celebritySaju.hourPillar) {
    if (userSaju.hourPillar === celebritySaju.hourPillar) {
      score += 10;
    }
  }
  
  return score;
}
```

### 3.3 데이터 스키마

#### 3.3.1 Celebrity 데이터 구조
```typescript
interface Celebrity {
  id: string;
  name: {
    ko: string;
    en: string;
    original?: string;  // 본명
  };
  birthDate: string;    // YYYY-MM-DD
  birthTime?: string;   // HH:mm (알려진 경우)
  category: 'idol' | 'actor' | 'singer' | 'comedian' | 'youtuber' | 'athlete';
  group?: string;       // 소속 그룹 (있는 경우)
  gender: 'male' | 'female';
  nationality: string;
  image: string;        // 이미지 파일명
  saju: {
    yearPillar: string;   // 년주 (예: "甲子")
    monthPillar: string;  // 월주
    dayPillar: string;    // 일주
    hourPillar?: string;  // 시주 (알려진 경우)
    dayMaster: string;    // 일간 (예: "甲")
    mainElement: string;  // 주요 오행 (예: "Wood")
  };
  popularity: number;   // 인기도 (정렬용, 1-100)
  hashtags: string[];   // 관련 해시태그
}
```
